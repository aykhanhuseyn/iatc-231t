## Declaration

`let` or `var` or `const`

| keyword    | scope      | reassign | redeclare | spoil global object |
| ---------- | ---------- | -------- | --------- | ------------------- |
| let        | block      | yes      | no        | no                  |
| const      | block      | no       | no        | no                  |
| var        | functional | yes      | yes       | yes                 |
| no_keyword | functional | yes      | yes       | yes                 |
