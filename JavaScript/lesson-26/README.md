# Function

## Syntax

```javascript
// named
function funcName() {
	console.log('hello');
}
// anonymous
function () {
	console.log('hello');
}

const funcName = function funcName() {
	console.log('hello');
}
const funcName = function () {
	console.log('hello');
}
```

### Function Declaration

```javascript
// function declaration
function funcName() {
	console.log('hello');
}
```

### Function Expression

```javascript
// function expression
const funcName = function () {
	console.log('hello');
};
```

### Arrow Function

```javascript
// arrow function
const funcName = (params) => {
	console.log(params);
};
```
